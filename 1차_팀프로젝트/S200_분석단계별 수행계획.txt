- 1단계 : 기초 Data 현황 파악 (Data 정의, type 등)

- 2단계 : 분석을 위한 데이터 정제 작업 실시
	- 결측치 확인 / Data Join / 신규 Column 생성 등

- 3단계 : 요구사항별 데이터 분석
            * 요건 정의
	   - 카드사용금액 : 월별 비교
   	   - 카드사용건수 : 요일 / 시간대별 비교
	   - 업종 분류 원칙 : 대분류 업종 (유의미한 결과가 필요 시, 소분류로 세분화 실시 예정)

               - 업종에 따른 요일별 카드 사용금액 분석
  	     → 대분류 업종별 카드 사용금액 분석 및 상위 2~3개 업종 추출
	     → 상위 2~3개 업종 코드의 세분화 코드별 사용금액 분석
               - 요일별 카드 이용 시간대 분석
  	     → 이용 시간대 분류 (새벽 / 오전 / 점심 / 오후 / 저녁)
               - 업종에 따른 요일별 카드 사용 시간대 분석
  	     → 상동
